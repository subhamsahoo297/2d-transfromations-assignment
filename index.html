2D Geometric Transformations

Drag the red points on the Identity shape (top-left) to see how transformations affect the geometry in real-time.

Identity

Base geometric shape.

Draggable: Drag the vertices to change the shape.

Translation

Shifts the object along X and Y axes.

Rotation

Rotates the object about the origin (0,0).

Scaling

Resizes the object. Negative flips it.

Shearing (Skew)

Slants the object along the axes.

Reflection

Flips the object across the chosen axis.
